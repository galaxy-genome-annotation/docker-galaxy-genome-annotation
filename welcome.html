Dockerized Galaxy + Apollo is running!


To get started, upload a fasta genome, and some gff3 formatted datasets. The JBrowse tool will let you build up a Jbrowse instance which is viewable inside of Galaxy.

Once you've done that, you'll need to register an account with Apollo. Finally, you can Create or Update an organism will then let you push the JBrowse instance into Apollo and associate it with an organism.
